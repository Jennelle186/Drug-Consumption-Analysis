**Illegal Drug Consumption Analysis**

**_Overview_**

Analyzed a large-scale survey dataset (2,000+ responses) to identify patterns and predictors of illegal drug consumption using data analysis and machine learning techniques

_**What I Did**_

  Cleaned, processed, and validated psychometric and demographic data for modeling results

  Applied Exploratory data analysis on behavior and socio-demographic variables

  Developed machine learning models for estimating drug use and major risk factors

  Developed distinct and decision-aiding visualization representations

_**Data Used**___

  Personality and behavioral assessments (NEO-FFI-R, BIS-11, ImpSS)

  Demographic variables (age, gender, education, ethnicity, location)

_**Tools & Skills**_

  Python, Pandas, NumPy

  Scikit-learn (machine learning)

  Matplotlib, Seaborn (data visualization)

  Jupyter Notebook
